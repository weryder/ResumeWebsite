Nucleus Annotation WebApp for Leica Biosystems

This WebApp project was done in order to test the feasability of creating an application that could intake an image of skin cell tissue and using machine learning in order to draw annotations around the nuclei. The WebApp was built using a Flask (Python) backend and had an HTML frontend. Javascript was used to pass information from frontend to backend with JSON. Bokeh a data visualization library was used to create an interactive HTML chart. OpenCV was used for computer vision and creating the annotations with a nueral network model that was trained and provided by Leica. William Ryder was responsible for the creation of the Flask backend, the APIs used to transfer data,the script that would pull the coordinates from the XML files and plot them, and for the custom Bokeh tools that were written in Javascript.
